# BANCO DE DADOS

**Olá, seja bem vindo a este Banco de Dados!**

Antes de iniciar ao longo desse banco de dados, você deve saber que há uma barra de navegação principal, na parte superior desta janela. 

Por ela, você pode optar por Paciente, onde cadastra um novo, ou pesquisa por um já existente; Dados onde busca os voluntários cadastrados; e a aba Sobre, na qual se conhece sobre o LABORE, ou sobre o EMOTIV.

### Paciente

Onde se realiza as ações de cadastros e pesquisas dos pacientes.

- **Novo**

  Cadastro de um novo paciente, diretamente ao banco de dados;

- **Pesquisar**

  Visualização dos resultados do EEG de cada voluntário

- **Fechar**

  Fechamento da tela do banco de dados.

### Dados

Visualização dos dados que estão no banco de dados.

- **Voluntários**

  Acompanhamento de todos os dados referente aos voluntários, dados estes que estão no banco de dados.

### Sobre

Pequenas informações sobre o local de estudo e o EEG utilizado.

- **LABORE** 

  Informações sobre o Laboratório onde foi desenvolvido o estudo;

- **EMOTIV EPOC**

  Informações sobre o EEG de baixo custo utilizado ao longo deste estudo.
